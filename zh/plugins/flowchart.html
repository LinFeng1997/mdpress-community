      # 安装

      yarn add mdpress-plugin-flowchart -D

      # 用法

      // .mdpress/config.js
module.exports = {
  plugins: ["flowchart"],
};

      # 语法

      @flowstart [preset]

<!-- 你的流程图代码 -->

@flowend

      mdpress-plugin-flowchart 依赖 flowchart.js。

      可用的预设配置有：

      • mdpress （默认）
      • ant

      参考当前可用的预设配置, feel free to submit your own preset. : )
